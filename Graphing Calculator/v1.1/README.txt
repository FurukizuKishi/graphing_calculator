To run the program:
- Install Portable Python 3.2.5.1 if it is not already installed. (http://portablepython.com/wiki/Download/)

Using Portable Python 3.2.5.1:
- Open IDLE-Portable.exe.
- Navigate to the 'Graphing Calculator' folder.
- Open the 'Program' folder and open Menu.py.
- Run using F5 or Run->Run Module.
